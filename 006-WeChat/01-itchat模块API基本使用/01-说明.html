Python操作微信

使用Python玩转微信

话题引入

生活工作中，肯定遇到过对方将微信消息撤销的情况

那么 问题来了，怎样才能知道，刚刚的消息到底是什么呢？难道逼他不成

下面由“师傅”给大家一一道来…...

有了 itchat， python 调用微信个人号从未如此简单

itchat 是一个开源的微信个人号接口

使用不到三十行的代码，你就可以完成一个能够处理所有信息的微信机器人。

当然，该 api 的使用远不止一个机器人，更多的功能等着你来发现。

如今微信已经成为了个人社交的很大一部分，希望这个项目能够帮助你扩展你的个人的微信号、方便自己的生活。

安装

可以通过本命令安装 itchat ：

Python2使用如下命令

pip install itchat

Python3使用如下命令

pip3 install itchat

简单的demo

有了 itchat ，如果你想要回复发给自己的文本消息，只需要这样：

import itchat

@itchat.msg_register(itchat.content.TEXT)
def text_reply(msg):
    itchat.send(msg['Text'], msg['FromUserName'])

itchat.auto_login()
itchat.run()

在弹出二维码之后，登录即可
