在后台中，显示出当前用户的信息，用表格的方式显示

2.1. 分析

如果想实现用户的统计信息，那么就需要2个基本的步骤

  1. 查询数据库所有用户的信息，或者分页，刚开始的时候不要想得过于复杂，直接查询所有就好
  2. 在已经下载的html模板中找一个html的标签，能够满足自己的需求即可

2.2. 查询数据库

因为需要从user表中查询，所以就需要在models.py中，定义User模型类了，试想一下，真的需要重新定义么？

之前在做前台的时候，已经定义过User模型类，那么就copy过来就好了，哈哈哈哈

完善main.py

from flask import Flask
from flask import render_template
from sqlalchemy.orm import sessionmaker

from models.models import User, engine

app = Flask(__name__)


@app.route("/")
def index():
    return render_template("index.html")


@app.route("/user_info")
def user_info():
    # 创建session对象
    DBSession = sessionmaker(bind=engine)  # 创建与数据库的会话，返回的是一个类
    session = DBSession()  # 生成链接数据库的实例
    # 查询user表中所有的用户数据
    all_user_info = session.query(User).all()
    print(all_user_info)
    # 关闭session
    session.close()
    return render_template("index.html")


if __name__ == '__main__':
    app.run()

2.4. 将查询出来的数据替换到模板中

模板中，使用循环

因为本案例中，需要从数据库中查询所有的信息，即查出来的是一个列表，

那么以为这列表中有多少个信息，就对应着上图中的表格有多少行信息，

所以，在模板中用如下的方式进行替换数据

{% for user_info in user_infos %}
<tr>
    <td>{{ user_info.user_id }}</td>
    <td>{{ user_info.user_name }}</td>
    <td>{{ user_info.head_img}}</td>
    <td>{{ user_info.short_description }}</td>
</tr>
{% endfor %}
