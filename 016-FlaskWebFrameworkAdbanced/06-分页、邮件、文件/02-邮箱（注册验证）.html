验证注册时的邮箱

1. 目的

实际工作中，为了验证用户注册的有效性，往往需要验证其电子邮箱(email)

本节的目的：用户注册后，向用户的email中发送一封邮件，等用户点击验证后，允许其登录

2. 发送邮件测试

安装：

pip3 install flask-mail

实际企业中，使用的是付费的邮箱，这样能够尽量避免发送失败等

本节知识，我们为了学习，所以使用的是163的邮箱作为发送者

注意：

注册163邮箱之后，去设置 -> POP3/SMTP/IMAP(在右侧导航栏)，然后开启你的SMTP服务，这时候会让你设置客户端授权码，这个授权码是重点，一定要记住

类型 服务器名称 服务器地址 SSL协议端口号 非SSL协议端口号
发件服务器 SMTP smtp.163.com 465/994 25

测试

from flask import Flask
from flask_mail import Mail, Message


app = Flask(__name__)
# 下面是SMTP服务器配置
app.config['MAIL_SERVER'] = 'smtp.163.com'  # 电子邮件服务器的主机名或IP地址
app.config['MAIL_PORT'] = 25  # 电子邮件服务器的端口
app.config['MAIL_USE_TLS'] = True  # 启用传输层安全
# 注意这里启用的是TLS协议(transport layer security)，而不是SSL协议所以用的是25号端口
app.config['MAIL_USERNAME'] = 'dong4716138_10@163.com'  # 你的邮件账户用户名
app.config['MAIL_PASSWORD'] = 'python123'  # 邮件账户的密码,这个密码是指的授权码!授权码!授权码!

mail = Mail(app)


@app.route('/')
def index():
    msg = Message('你好', sender='dong4716138_10@163.com', recipients=['993484988@qq.com'])
    # 这里的sender是发信人，写上你发信人的名字，比如张三。
    # recipients是收信人，用一个列表去表示。
    msg.body = '你好'
    msg.html = '<b>你好</b> stranger'
    mail.send(msg)
    return '<h1>邮件发送成功</h1>'


if __name__ == '__main__':
    app.run(debug=True)

值得注意的一点是，如果你是刚刚创建的163的邮箱，你最好先用163邮箱发送一封邮件，因为你在发送第一封邮件的时候，会让你设置发件人名字，如果不设置的话，你的邮件会被退回。

3. 注册时发送email

3.1 思路

  1. 修改数据库(模型)，添加2个字段
    1. 一个字段用来存储用户的等级，例如默认是0表示未进行邮箱验证，1表示邮箱验证成功
    2. 另外一个字段用来存储一个加密字符串，并且将这个加密字符串通过email发送给用户，当用户从email中点击后将用户的等级由0改为1，从而完成用户有限验证的步骤
  2. 用户点击注册时，要发送email且带有超链接，目的是让用户点击进行确认
  3. 后端登录的视图函数中判断用户是否已经验证过邮箱，即等级是否是1，如果不是则提醒用户进行邮箱验证
