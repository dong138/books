文件上传

1. 目的

很多时候，需要上传文件，例如头像，附件等等 其实都用到文件上传

本节目的，利用文件上传，将头像进行修改

2. 思路分析

  1. 在profile页面中，整个可以点击的超链接
  2. 点击那个超链接后，能够跳转到一个新的页面，这个页面用来显示之前用户的数据
  3. 在新的页面中，整一个form表单，里面有各种input，其中一个的类型是file
  4. 点击submit的input按钮后，能够用post请求将数据提交后后端，即用一个视图函数来处理这个请求
  5. 在视图函数中获取用户修改的各个数据，然后通过request.file.get("input为了file的name值")提出出文件，然后使用save()保存到一个路径即可

@index_blu.route("/edit", methods=["GET", "POST"])
@login_user_data
def edit():

    if not g.user:
        return redirect(url_for(".login"))

    if request.method == "GET":
        return render_template("index/edit.html", user=g.user)
    elif request.method == "POST":
        # 1.提取数据
        # user_id = request.form.get("user_id")
        username = request.form.get("username")
        password = request.form.get("password")
        email = request.form.get("email")
        # image = request.form.get("image")
        content = request.form.get("content")

        current_user = g.user

        print(current_app.root_path)  # /Users/wangmingdong/Desktop/个人简历7/app

        # 获取头像
        print(request.files)
        f = request.files.get('image')
        if f:
            # basepath = os.path.dirname(__file__)  # 当前文件所在路径
            # # 注意：没有的文件夹一定要先创建，不然会提示没有该路径
            image_type = f.filename[f.filename.rfind("."):]
            name_hash = hashlib.md5()
            name_hash.update(f.filename.encode("utf-8"))
            image_file_name = name_hash.hexdigest()
            image_file = image_file_name + image_type
            image_path = os.path.join('/static/upload/images', image_file)
            upload_path = os.path.join(current_app.root_path, 'static/upload/images', image_file)
            print(upload_path)
            f.save(upload_path)
            current_user.head_img = image_path

        # 修改用户信息
        # current_user.user_id = user_id
        current_user.user_name = username
        current_user.password = password
        current_user.email = email
        # current_user.head_img = image
        current_user.short_description = content

        # 提交数据
        db.session.commit()

        return redirect(url_for(".edit"))
