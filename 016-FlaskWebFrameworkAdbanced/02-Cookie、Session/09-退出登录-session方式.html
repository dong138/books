退出登录

1. 引入

正常情况下，关闭浏览器不会将cookie等信息清除，如果真的要清除：

  1. 浏览器当达到一定缓存量之后，自动清除
  2. 手动进行清除

所以，通常情况下，即使管理浏览器你的登录信息依然会保存在浏览器中，因此才能实现当下一次再访问同一个网站的时候，虽然没有重新登录，但是依然能够看到登录后才能看到的信息，真是个利用了这个特点

2. 退出

如果要真正的退出登录，目的只有1个：清除浏览器中缓存的cookie信息

问题是，总不能每次都需要手动清除cookie来退出吧，因此一个完整的网站中，需要有退出的功能

直白点说：在大部分的页面中有个超链接，这个超链接连接到一个视图函数，这个视图函数中有清除session的功能

3. 实现

views/index/views.py中添加如下视图函数

@index_blu.route("/logout")
def logout():
    """
    退出登录视图函数
    :return:
    """
    session.clear()  # 清除所有的session
    return redirect("/")
