OpenCV基础

图片基本操作

1. 引入

OpenCV操作图片是很方便的，能够对图像更进一步处理，例如合并图片、添加水印等

OpenCV中如果要操作图片，有几个很简单的方法:

  • cv2.imread()
  • cv2.imshow()
  • cv2.imwrite()

2. 操作图片

2.1 读取图像

使用cv2.imread()函数读取图像

注意点1：

图像应该在和我们文件一起的目录中，或者在程序里应该给出图像的完整路径。

注意点2：（指定图像读取方式的flag）

  • cv2.IMREAD_COLOR：加载一个彩色图像，图像的任何透明度都将被忽略。这是默认标志
  • cv2.IMREAD_GRAYSCALE：以灰度模式 (grayscale mode) 加载图像
  • cv2.imread_same：加载图像，包括alpha channel

小窍门：

我们除了输入上面三个中的flag ，我们也可以用1，0，-1来代替

示例代码

import cv2  # 导入cv2 也就是opencv-python 包

# 导入灰度模式的彩色图片一张
img = cv2.imread('dog.jpeg', 0) # 我已经将照片放入和我的python 文件一起的文件夹了；0代表灰度模式

怎么查看图片放没放对位置呢？可以接着输入

print(img)

如果结果输出是一大段矩阵，比如

[[139 138 139 ... 186 188 188]
 [136 137 138 ... 188 188 201]
 [134 135 136 ... 192 202 189]
 ...
 [ 54  54  54 ... 102  75  60]
 [ 54  55  55 ...  81  64  57]
 [ 54  55  55 ...  69  55  58]]

Process finished with exit code 0

那么就说明图片没错，pycharm 读取图片成功了；如果显示的是none，说明图片哪儿出错了，有可能是路径或则别的。

2.2 显示图像

使用cv2.imshow()函数在窗口中显示图像

窗口会自动适合于图像的大小

示例代码1

cv2.imshow('dog',img)  # 第一个参数是窗口名称，第二个参数是图像
cv2.waitKey(0)  # 键盘绑定函数，等待按键
cv2.destroyAllWindows() # 按下键盘中的按键，启动该程序销毁所有窗口

说明

  1. 第一个参数是窗口名称，它是一个字符串
  2. 第二个参数是我们的图像
  3. 我们可以创建任意多个窗口，但窗口名称需要保证不同
  4. cv2.waitKey()是一个键盘绑定函数，它的参数是以毫秒为单位。该函数等待在指定毫秒内任何的键盘事件(比如按下键盘中任意一个键)。如果在那个时间段内按下键盘中任何键，程序将继续。如果参数设置为了0，它将无限期地等待键击。它还可以设置为检测特定的击键，如按下a键等，我们将在后面讨论。
  5. destroyallwindows()函数用来销毁我们创建的所有窗口(虽然这里只有一个)。如果想要销毁任何特定的窗口，我们可以使用函数cv2.destroyWindow()，在其中传递确切的窗口名称作为参数。注意，有一种特殊的情况，我们可以先创建窗口再将图像加载到其中。在这种情况下，我们可以指定窗口是否可调整大小。它是通过函数cv2.namedWindow()完成的。默认情况下，flag是cv2.WINDOW_AUTOSIZE；但是如果我们指定flag为cv2.WINDOW_NORMAL，我们就可以调整窗口大小了。当图像尺寸过大时，在窗口中添加这种跟踪条track bar 是很有帮助的

示例代码2

# 可以拖拽边框，进行缩放
cv2.namedWindow('momo', cv2.WINDOW_NORMAL)
cv2.imshow('momo',img)
cv2.waitKey(0)
cv2.destroyAllWindows()

2.3 保存图像

使用函数cv2.imwrite()来保存图像

示例代码

cv2.imwrite('black_white_dog.jpg', img)  # 保存的文件名字，要保存的图像目表

说明

  1. 第一个参数是文件名
  2. 第二个参数是要保存的图像

运行效果

这将把图像以jpg格式保存在工作目录中。

合并之后的代码

下面这个程序将加载一个灰度图像，显示它，如果你按退出ESC 键将直接退出不保存，如果你按s 键，将会保存图像再退出

# coding=utf-8
import cv2

# 1. 读取图片
img = cv2.imread("dog.jpeg", 0)

# 2. 看一下读取的图片数据
print(img)

# 3.1 命名一个创建，并设置可以自动缩放
cv2.namedWindow("dog", cv2.WINDOW_NORMAL)
# 3.2 在刚刚的窗口中显示 图片数据
cv2.imshow("dog", img)

# 4. 等待按下任意键
cv2.waitKey(0)

# 5. 保存灰度图片
cv2.imwrite("black_white_dog.jpg", img)

# 6. 关掉所有窗口
cv2.destroyAllWindows()

注意

如果你使用的是64位计算机，那么你需要修改k = cv2.waitKey(0) 才行；修改成如下所示：k = cv2.waitKey(0) & 0xFF

3. 访问图像属性

图像属性包括：

  • 行数
  • 列数
  • 通道数channels
  • 图像数据类型
  • 像素数
  • 。。。等等。。。

图像的形状可以由img.shape 来得到，它会返回以元组为形式的的行数，列数和通道数(如果图像是彩色的)：

print(img.shape)

# 得到结果如下
(540, 720, 3)  # 图像的行数、列数、通道数

Process finished with exit code 0

注意：

如果图像不是彩色的是灰度的，返回的元组就只会包含行数和列数，没有通道数

因此，用shape 来检测加载的图像是灰度图像还是彩色图像是一种很实用的方法。

像素的总数(行数x列数x通道数)可以用img.size来得到：

print(img.size)

# 得到结果如下
1166400

Process finished with exit code 0

图像的数据类型可以通过img.dtype来获得：

print(img.dtype)

# 得到结果如下
uint8

Process finished with exit code 0

img.dtype 在debug 过程中是非常重要的，因为OpenCV-Python 代码中的大量错误都是由无效的数据类型引起的。
