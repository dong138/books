步骤3（完善run方法）

  1. 链接tcp服务器
  2. 采集摄像头数据
  3. 转换为指定的大小，例如640x480
  4. 压缩图片
  5. 发送图片
    1. 为了能够在接收方接收这个数据时，完整接收完这一帧数据（即一副图片的完整数据）再显示，需要发送时额外添加一个这个图片大小的说明
    2. 使用struct.pack进行组成特定长度的数据信息
      1. pack中i表示 4个字节固定长度
      2. pack中h表示 2个字节固定长度
import socket
import struct
import time
import cv2
import numpy


class Camera(object):
    def __init__(self, dest_ip, dest_port):
        # 在这里应该完成
        # 1. 使用socket创建套接字
        # 2. 使用cv2打开摄像头
        self.resolution = (640, 480)  # 分辨率
        # 创建tcp套接字
        self.client_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        self.dest_info = (dest_ip, dest_port)
        # 打开默认的第1个摄像头
        self.camera = cv2.VideoCapture(0)

    def run(self):
        # 在这里应该完成
        # 1. 链接tcp服务器
        # 2. 获取摄像头的数据
        # 3. 发送到tcp服务器
        # ----------- 添加的代码开始 ----------------
        # 链接tcp服务器
        self.client_socket.connect(self.dest_info)

        try:
            while True:
                time.sleep(0.1)  # 推迟线程运行 0.1s (即意味着采集数据，慢一点)
                _, img = self.camera.read()  # 读取摄像头的一帧数据

                # 将元摄像头采集到的数据进行调整大小，例如调整为640x480
                # resolution必须为元组
                img = cv2.resize(img, self.resolution)

                # 为了能够将图片通过网络发送的时候快速的发送过去，需要将图片进行压缩之后再发送，对方接收到数据之后需要先解压，然后处理
                # (1) 压缩图片 第1个参数 表示压缩的格式，它会返回一个元组（第2个元素就是压缩之后的数据）
                _, img_encode = cv2.imencode('.jpg', img)

                # (2) 转换为numpy格式数据
                img_code = numpy.array(img_encode)

                # (3) 生成相应的bytes类型数据，其实就是二进制数据
                img_data = img_code.tostring()

                # 按照相应的格式进行打包发送图片
                self.client_socket.send(struct.pack("ihh", len(img_data), self.resolution[0], self.resolution[1]) + img_data)
        except KeyboardInterrupt:
            return
        # ----------- 添加的代码结束 ----------------

    def __del__(self):
        # 在这里应该完成(释放资源的功能)
        # 1. 关闭套接字
        # 2. 关闭摄像头
        pass


def main():

    # 1. 获取要链接的服务器ip地址
    dest_ip = input("请输入远程监控服务器的ip:")
    if not dest_ip:
        dest_ip = "127.0.0.1"
