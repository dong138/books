3.2. 安装flask

python3版本下安装flask-0.10.1的包 :

pip install 包名称

注意：

  • pip install 模块时，默认安装最新版本

  • 如果需要安装指定的版本，如下方式：（安装flask的1.0.2版本）

    pip install flask==1.0.2
