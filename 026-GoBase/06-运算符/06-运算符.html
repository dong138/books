在前面的课程中，我们已经学习了变量，常量，那么接下来我们学习一下运算符。

1. 什么是运算符

一句话描述：就是程序中的加减乘除等操作

GO语言中常见的运算符有

  • 算术运算符
  • 赋值运算符
  • 关系运算符
  • 逻辑运算符等

2. 算术运算符

2.2 算术表达式

由算术运算符连接起来的式子.如:

1+1  // 直接进行加法操作
a-b  // 变量ab前面已声明并赋初值，直接进行减法操作

看下面例子

package main

import "fmt"

func main()  {
    // 定义2个变量
    var a int = 5
    b := 2

    // 使用运算运算符操作a、b
    fmt.Println("a + b =", a+b)  // 加法
    fmt.Println("a - b =", a-b)  // 减法
    fmt.Println("a * b =", a*b)  // 乘法
    fmt.Println("a / b =", a/b)  // 取商
    fmt.Println("a % b =", a%b)  // 取余数
}

2.3 自增与自减操作

package main

import "fmt"

func main()  {
    // 定义一个变量，用来测试 ++、--
    a := 10
    fmt.Println("a =", a)  // 10

    a = a+1
    fmt.Println("a =", a)  // 11

    a ++
    fmt.Println("a =", a)  // 12

    a --
    fmt.Println("a =", a)  // 11
}

小结：

a++等价于a=a+1

注意点1：

package main

import "fmt"

func main()  {
    // 定义一个变量，用来测试 ++、--
    a := 10
    fmt.Println("a =", a)  // 10

    // 定义一个变量用来保存结果
    var b int

    // 保存a++的结果
    b = a ++

    // 测试a、b的值
    fmt.Println("a =", a, "b =", b)
}

注意2：

GO语言中没有前自增，这是与其它编程语言不同的地方，即不能出现++、--在变量名的前面(例如++a)

2.4 练习

某学生三门课成绩，语文、数学、英语，编程求总分和平均分.

package main

import "fmt"

func main()  {
    // 定义3个变量，分别存储语数外的成绩
    score_chinese := 95
    score_math := 94
    score_english := 93

    // 定义一个变量用来存储平均值
    var average int

    // 计算平均值
    average = (score_chinese + score_math + score_english) / 3

    // 打印平均值
    fmt.Println("平均值是", average)
}

虽然已经计算出总分与平均分了，但是我们可以对上面的程序进行优化，怎么优化呢？示例如下：

package main

import "fmt"

func main()  {
    // 定义3个变量，分别存储语数外的成绩
    score_chinese := 95
    score_math := 94
    score_english := 93

    // 定义一个变量用来存储平均值
    // var average int

    // 计算平均值
    // average = (score_chinese + score_math + score_english) / 3

    // 打印平均值
    // fmt.Println("平均值是", average)
    fmt.Println("平均值是", (score_chinese + score_math + score_english) / 3)
}

注意点：

  • 该程序比第一种实现减少了变量的定义，可以减少对内存的占用，所以变量不是定义越多越好。

  • 注意运算优先级：先乘除,后加减,有括号先算括号里的,相同级别的从左至右运算a=((1+5)+3)*2

  • 小括号可以无限制的套用,但一定要成对出现

3. 赋值运算符

关于赋值运算符前面我们已经使用过多次，赋值运算符=

var num int
num = 9
num = num + 1

这里的=号是赋值运算符,不是数学义意上的相等.

常见的赋值运算符如下，前面我们使用的=是普通赋值，+=，-=等我们称为复合赋值运算符

package main

import "fmt"

func main()  {
    a := 2
    b := 3
    c := 4
    c += a * b  // c = c + a * b = 4 + 2* 3 = 10
    fmt.Println("c =", c)

    a = 2
    b = 3
    c = 4
    c *= a + b  // c = c * (a + b) = 4 * (2 + 3) = 4 * 5 = 20
    fmt.Println("c =", c)
}

4. 关系运算符

关系运算符我们又称为比较运算符，听到比较这个词，大家想到的就是比较大小，例如：

大象的重量(1500)>老鼠的重量(1)

兔子的寿命(3)>乌龟的寿命(1000)

我的年龄(20)==你的年龄(20)

提问：

比大比小的结果是什么？

结果是对或者是错

在GO语言中，怎样表示对于错呢？

关系运算的结果是布尔类型的

bool值只有两个

  • 真:true

  • 假:false

GO语言中常见的关系运算符如下：

由以上关系运算符链接的表达式我们称为关系表达式

关系运算符具体使用方式如下：

package main

import "fmt"

func main()  {
    // 定义变量，用来存储布尔值
    var result bool

    // 大象的重量(1500)>老鼠的重量(1)
    result = 1500 > 1
    fmt.Printf("%t\n", result)

    // 兔子的寿命(3)>乌龟的寿命(1000)
    result = 3 > 1000
    fmt.Printf("%t\n", result)

    // 我的年龄(20)==你的年龄(20)    
    result =  20 == 20
    fmt.Printf("%t\n", result)
}
