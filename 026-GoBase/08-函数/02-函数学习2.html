5. 函数嵌套调用

5.1 什么是函数嵌套调用

所谓函数嵌套调用，就是函数执行代码的过程中又去调用另外一个函数

package main

import "fmt"


func test2() {
    fmt.Println("---test2---函数中的代码...")
}

func test1() {
    fmt.Println("---test1---函数中的代码...")
    test2()
}

func main()  {
    test1()
}

5.2 基本案例

需求：

计算3个数的平均值

参考代码1

package main

import "fmt"

func Avarage(temp int) int {
    return temp/3  // 返回平均值
}


func Add3Nums(a, b, c int) (sumResult int){
    sumResult = a + b + c  // 计算出3个数的和
    return  // 返回三个数的和
}

func main()  {
    var sumRet, avaRet int
    sumRet = Add3Nums(11, 22, 33)  // 用sumRet存储返回值
    fmt.Println("三个数的和是:", sumRet)  // 打印结果
    avaRet = Avarage(sumRet)  // 用avaRet存储返回值
    fmt.Println("三个数的平均值是:", avaRet)
}

图示说明

说明：

为了完成平均值的计算，整体划分为2部分

  1. 计算3个数的和，并将这部分功能放到单独的一个函数中来实现
  2. 计算一个数/3的结果，并将这部分功能放到单独的一个函数中来实现

然后在main函数中分别调用他们，先调用计算和的函数，然后把结果存储后 ，再调用除以3的函数 且把第一步的值传递过去 把除以3的结果返回

参考代码2

package main

import "fmt"

func Avarage(temp int) int {
    return temp/3  // 返回平均值
}


func Avarage3Nums(a, b, c int) (int){
    var sumResult int = a + b + c  // 计算出3个数的和
    return  Avarage(sumResult)  // 返回三个数的平均值
}

func main()  {
    var avaRet int
    avaRet = Avarage3Nums(11, 22, 33)  // 用avaRet存储返回值
    fmt.Println("三个数的平均值是:", avaRet)
}

图示说明

说明

此方法与上一种方法很不一样的地方在于：

第1种方式：main函数分别调用2个函数，且在调用第2个时将第1个函数的返回值当做参数传递

第2中方式：main函数只调用了1个函数，这个函数内部在完成3个数的求和之后，又调用了另外一个函数完成平均值的计算，然后把 最后的平均值的结果当做返回值给返回

5.3 案例2：不定参数函数嵌套调用

package main

import "fmt"

func Add(numsTemp ... int) int {
    var numsCount int
    var addRest int

    numsCount = len(numsTemp)

    for i:=1; i<=numsCount; i++ {
        addRest += numsTemp[i-1]
    }
    return addRest
}

func Mul(numsTemp ... int) int {
    var numsCount int
    var mulRest int = 1  // 注意这里，要让默认值是1，如果是0的话乘以任何数都是0

    numsCount = len(numsTemp)

    for i:=1; i<=numsCount; i++ {
        mulRest *= numsTemp[i-1]
    }
    return mulRest
}


func Test(nums ... int) (int, int){

    var ret1, ret2 int

    // 调用函数 计算相加的结果
    ret1 = Add(nums ...)

    // 调用函数 计算相减的结果
    ret2 = Mul(nums ...)

    // 返回结果
    return  ret1, ret2
}

func main()  {
    var addRet, sumRet int
    addRet, sumRet = Test(2, 2, 3)
    fmt.Printf("相加结果是:%d, 相乘结果是:%d\n", addRet, sumRet)
}

6. 函数类型

在讲解函数类型之前，先简单的回顾一下前面怎样定义一个函数以及怎样调用一个函数的。

package main

import "fmt"

func Test() {
    fmt.Println("----------------------")
    fmt.Println("    hello  world")
    fmt.Println("----------------------")
}

func main()  {
    Test()
}
  • 先要定义函数
  • 然后再调用函数
  • 函数可以有参数，也可没有参数
  • 函数可以有返回值，也可以没有返回值
  • 普通参数
  • 不定参数列表

在GO语言中还有另外一种定义使用函数的方式,就是函数类型

所谓的函数类型就是将函数作为一种类型可以用来定义变量，这种用法类似于前面我们讲过的int ,float64,string等类型，这些类型都是可以用来定义变量。

函数类型demo

package main

import "fmt"

func Test(name string, age int) string {
    fmt.Println("----------------------")
    fmt.Printf("姓名:%s\n", name)
    fmt.Printf("姓名:%d\n", age)
    fmt.Println("----------------------")
    return "ok"
}

type NewTest func(string, int) string

func main()  {

    var xx NewTest
    xx = Test

    // 使用普通的方式调用Test函数
    fmt.Println(Test("东哥", 18))

    // 使用xx调用Test函数
    fmt.Println(xx("王老师", 30))

}

说明

这是我们在GO语言中使用函数类型的方式完成函数的调用，可能有同学感觉这种方式比较麻烦，不如我们前面讲解的方式使用起来简单。那么在这里，要求大家先理解这种方式，能记住语法就可以了。关于该方式的优势，等我们讲解完面向对象编程这个知识点后，大家就有深刻的理解了。

7. 函数作用域

7.1 局部变量

前面我们定义的函数中，都经常使用变量。那么我们看一下如下程序的输出结果：

package main

import "fmt"

func Test() {
    a := 5
    a += 1
}

func main()  {
    a := 9
    Test()
    fmt.Println(a)
}

最终的输出结果是9，为什么呢？

在执行fmt.Println(a)语句之前，我们已经调用了函数Test(),并在该函数中我们已经重新给变量a赋值了。但是为什么结果没有发生变化呢？

原因：这就是变量的作用范围（作用域）的问题。

在Test( )函数中定义的变量a,它的作用范围只在改函数中有效，当Test( )函数执行完成后，在该函数中定义的变量也就无效了。也就是说，当Test( )函数执行完以后，定义在改函数中所有的变量，所占有的内存空间都会被回收。

所以，我们把定义在函数内部的变量称为局部变量

局部变量的作用，为了临时保存数据需要在函数中定义变量来进行存储，这就是它的作用。

并且，通过上面的案例我们发现：

不同的函数，可以定义相同的名字的局部变量，但是各用个的不会产生影响。

例如：我们在main( )函数中定义变量a,在Test( )函数中也定义了变量a，但是两者之间互不影响，就是因为它们属于不同的函数，作用范围不一样，在内存中是两个存储区域。

7.2 全局变量

有局部变量，那么就有全局变量。

所谓的全局变量：既能在一个函数中使用，也能在其他的函数中使用，这样的变量就是全局变量

定义全局变量的特点：

  • 定义在函数外部的变量就是全局变量
  • 全局变量在任何的地方都可以使用

全局变量案例

package main

import "fmt"

// 定义一个全局变量
var a int

func Test() {
    a = 5
    a += 1
}

func main()  {
    a = 9
    Test()
    fmt.Println(a)
}

注意：

在上面的案例中，我们在函数外面定义了变量a,那么该变量就是全局变量，并且Test( )函数和main( )函数都可以使用该变量。

该程序的执行流程是：

先执行main( )函数，给变量a赋值为9,紧接着调用Test( )函数，在改函数中完成对变量a的修改。

由于main( )函数与Test( )函数所使用的变量a是同一个，所以当Test( )函数执行完成后，变量的a已经变成了6 ，回到main( )函数执行后面的代码，也就是fmt.Println(a),输出的值就是6

7.3 全局变量与局部变量名字相同问题

可能有同学已经发现该程序和我们前面写的程序还有一点不同的地方是：

第一个程序我们是a:=9,但是第二个程序执行修改成了 a=9,

为了验证，此类问题，我们修改代码如下：

该程序与上面的程序不同之处在于，该程序是a:=9,上面的程序是a=9

现在大家思考一下该程序的结果是多少？

最终结果是9

原因是：

a := 9

// 等价于
var a int
a = 9

也就是定义一个整型变量a,并且赋值为9

那么现在的问题是

全局变量a, 与main函数中的局部变量a 重名，此时会怎样呢？

当全局变量与局部变量名称一致时，局部变量的优先级要高于全局变量。

所以在main( )函数中执行fmt.Println(a)时输出的是局部变量a的值。

但是Test( )函数中的变量a还是全局变量

注意

  • 以后在开发中，尽量不要让全局变量的名字与局部变量的名字一样

总结

  1. 在函数外边定义的变量叫做全局变量
  2. 全局变量能够在所有的函数中进行访问
  3. 如果全局变量的名字和局部变量的名字相同，那么使用的是局部变量的，小技巧强龙不压地头蛇

8. 递归函数

8.1 什么是递归函数

在一个函数在内部如果又调用了是自己本身的话，这个函数就是递归函数

8.2 案例

package main

import "fmt"

func Test(temp int) {

    fmt.Println("111 temp=", temp)

    if temp == 1 {
        return
    }

    Test(temp - 1)
    fmt.Println("222 temp=", temp)

}

func main()  {
    Test(3)
}

注意：递归函数的执行流程

递归函数的作用：

  • 计算阶乘n! = 1 * 2 * 3 * ... * n
  • 电商网站中的商品类别菜单的应用
  • 查找某个磁盘下的文件

9. 匿名函数

匿名函数就是没有名字的函数

package main

import "fmt"


func main()  {
    // 定义一个变量用来存储一个函数，也就是说f就是一个函数
    f := func() {
        fmt.Println("我是一个函数的执行代码...1")
        fmt.Println("我是一个函数的执行代码...2")
        fmt.Println("我是一个函数的执行代码...")
    }

    // 因为f不是普通的类型变量，而是存储的函数，所使用时需要添加()
    f()
}
