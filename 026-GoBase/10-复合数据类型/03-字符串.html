Go基础

字符串

1. 为什么要学习字符串操作

在后面的学习中我们会学习文件，文件的作用就是把数据存储到硬盘的方式

很多时候当我们从文件中将数据读取出来以后，并不是直接将数据打印出来，而是要做相应的处理。例如：去掉空格等一些特殊的符号，对一些内容进行替换等。

此时就涉及到对一些字符串的处理。在对字符串进行处理时，需要借助于包strings

2. 常用的字符串处理函数

2.1 Contains

func Contains(s, substr string) bool

说明

  • 字符串s中是否包含substr，返回bool值

demo

package main

import "fmt"
import "strings"

func main()  {
    var str string = "hello world"
    fmt.Println(strings.Contains(str, "he"))  // true
    fmt.Println(strings.Contains(str, "Wo"))  // false
}

运行效果

func Join(a []string, sep string) string

2.5 Replace

func Replace(s, old, new string, n int) string

说明

  • 在s字符串中，把old字符串替换为new字符串，n表示替换的次数，小于0表示全部替换

demo

package main

import "fmt"
import "strings"

func main()  {
    s := "gogogo"
    ret := strings.Replace(s, "o", "O", 2)
    fmt.Println(ret)
    ret = strings.Replace(s, "o", "O", -1)
    fmt.Println(ret)
}

运行效果

3. 字符串转换

通过以上的讲解，发现字符串的处理是非常重要的

GO语言也提供了字符串与其它类型之间相互转换的函数

相应的字符串转换函数都在strconv包中

3.1 Format

Format系列函数把其他类型的转换为字符串

将bool值转换为字符串

package main

import "fmt"
import "strconv"

func main()  {
    var str string
    str = strconv.FormatBool(false)
    fmt.Println(str)
    fmt.Printf("%T", str)
}

将int值转换为字符串

package main

import "fmt"
import "strconv"

func main()  {
    var str string
    str = strconv.FormatInt(100, 10)  // 第1个参数是要转换的值，第2个参数是进制 10表示十进制
    fmt.Println(str)
    fmt.Printf("%T", str)
}

将float值转换为字符串

package main

import "fmt"
import "strconv"

func main()  {
    var str string
    // 'f'值打印的格式为小数方式，3指的是小数点位数，64表示以64位处理
    str = strconv.FormatFloat(3.14, 'f', 3, 64)
    fmt.Println(str)
    fmt.Printf("%T", str)
}

3.2 Parse

Parse系列函数把字符串转换为其他类型

package main

import "fmt"
import "strconv"

func main()  {
    // 字符串 ---> bool
    var flag bool
    flag, _ = strconv.ParseBool("true")
    fmt.Println(flag)
    fmt.Printf("%T", flag)

    // 字符串 ---> int
    var num int64
    num, _ = strconv.ParseInt("100", 10, 64)
    fmt.Println(num)
    fmt.Printf("%T", num)
}
