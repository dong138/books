Map

1. 引入

前面我们学习了GO语言中数组，切片类型，但是我们发现使用数组或者是切片存储的数据量如果比较大，那么通过下标来取出某个具体的数据的时候相对来说，比较麻烦。例如：

package main

import "fmt"

func main()  {
    // 定义一个切片，用来存储学生成绩
    scores := []int{100, 99, 98, 100, 87}

    // 定义变量用来存储学生的序号，即下标
    zhangsan := 0
    lisi := 1
    wagnwu := 2
    maliu := 3
    qianqi := 4

    // 获取学生分数
    fmt.Println("张三的分数:", scores[zhangsan])
    fmt.Println("李四的分数:", scores[lisi])
    fmt.Println("王五的分数:", scores[wagnwu])
    fmt.Println("马六的分数:", scores[maliu])
    fmt.Println("钱七的分数:", scores[qianqi])
}

运行效果

有没有一种结构能够帮我们快速的操作数据呢？

答案：Map

2. 什么是Map

Map 是一种无序的键值对的集合，是由键和值构成的

我们可以把Map理解为映射，即通过键来操作值

怎样来理解映射呢？打个比方古时候有烽火台，只要有火，就说明有敌情，此时我们把火---->敌情

同理在学习数组以及切片时，我们要操作元素，是通过下标实现的，此时可以把下标---->元素

但是我们知道下标必须是从0开始，而且全部都是数组，当数量少时还好，但是数量稍稍多一点，就感觉复杂

此时我们通过Map，就可以做到字符串---->元素，这样就非常方面操作元素了

3. 定义格式

var test map[键的类型]值的类型

demo

var test map[string]int

说明

  • 定义了一个Map，键的类型是string，值的类型是int

4. 案例:基本使用

package main

import "fmt"

func main()  {
    // 定义Map
    var scores map[string]int = map[string]int{"zhangsan":100, "lisi":99, "wangwu":100, "maliu":86, "qianqi":99}

    // 获取学生分数
    fmt.Println("张三的分数:", scores["zhangsan"])
    fmt.Println("李四的分数:", scores["lisi"])
    fmt.Println("王五的分数:", scores["wangwu"])
    fmt.Println("马六的分数:", scores["maliu"])
    fmt.Println("钱七的分数:", scores["qianqi"])
}

运行效果

说明

  • 发现某个键key都对应的一个值value，如果现在要查询某个值，直接根据键就可以查询出某个值

7. Map是无序的

package main

import "fmt"

func main()  {
    // 定义Map
    var scores map[string]int = map[string]int{"zhangsan":100, "lisi":99, "wangwu":100, "maliu":86, "qianqi":99}

    // 使用for循环遍历Map
    for key, value := range scores{
        fmt.Println("key =", key, " value =", value)
    }

}

运行效果

8. 注意点

8.1 key不能相同

  • 在字典中的键是不允许重复的，就像身份证号一样 是唯一的

8.2 对Map不能用cap，但可以用len

package main

import "fmt"

func main()  {
    // 定义Map
    var scores map[string]int = map[string]int{"zhangsan":100, "lisi":99, "wangwu":100, "maliu":86, "qianqi":99}

    fmt.Println(len(scores))
}

  • 字典中不能使用cap函数，只能使用len( )函数。len( )函数返回Map拥有的键值对的数量

8.3 Map 作为函数参数是引用传递

package main

import "fmt"


func Test(temp map[string]int)  {
    temp["wangwu"] = 66
}


func main()  {
    // 定义Map
    var scores map[string]int = map[string]int{"zhangsan":100, "lisi":99, "wangwu":100, "maliu":86, "qianqi":99}

    fmt.Println(scores)

    Test(scores)

    fmt.Println(scores)
}
