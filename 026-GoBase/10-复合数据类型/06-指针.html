Go基础

指针

1. 变量内存与地址

前面我们讲过存储数据的方式，可以通过变量，或者复合类型中的数组，切片，Map,结构体。

我们不管使用变量存储数据，还是使用符合类型存储数据，都有两层的含义：

  • 存储的是什么
  • 存储到哪里

接下来，通过变量来说明以上两个含义。

package main

import "fmt"

func main()  {

    var i int = 100

    fmt.Println("i =", i)
    fmt.Println("i的地址 =", &i)

}

说明

  • 第一个Println( )输出变量i的值，这个实际上就是输出内存中存储的数据。
  • 第二个Println( )输出的是变量i在内存中的地址。

通过如下图来给大家解释：

怎样理解100与0xc000016070？

  • 100就是存储的数据
  • 0xc000016070就是存储数据的内存序号

这个内存地址和实际生活中的地址也很相似

例如：大家可以将内存空间想象成，我们上课的教室，教室中存放有学生，那么现在要找一个学生，必须要知道具体的地址以及教室门牌号。

2. 指针变量

现在已经知道怎样获取变量在内存中的地址，但是如果想将获取的地址进行保存，应该怎样做呢？

可以通过指针变量来存储

所谓的指针变量：就是一个变量它存储的数据不是普通的100、200等，而是一个地址

3. 指针变量的定义

基本格式

var 变量名 *类型

demo

package main

import "fmt"

func main()  {

    // 定义普通的变量
    var i int = 100

    // 定义指针变量
    var p *int
}

说明

  • 指针变量p的定义是通过*这个符号来定义，类型为*int
  • 表示存储的是一个整型变量的地址

注意

  • 如果指针变量p存储的是一个字符串类型变量的地址，那么指针变量p的类型为*string

4. 通过指针赋值、取值

4.1 取值

package main

import "fmt"

func main()  {

    // 定义普通的变量
    var i int = 100

    // 定义指针变量
    var p *int

    // 给指针变量赋值
    p = &i

    // 获取100
    // 通过变量i获取
    fmt.Println(i)
    fmt.Println(*p)

}

说明

  • p=&i 将变量i的地址取出来，并且赋值给指针变量p，也就是指针变量p指向了变量i的存储单元

4.2 赋值

package main

import "fmt"

func main()  {

    // 定义普通的变量
    var i int = 100

    // 定义指针变量
    var p *int

    // 给指针变量赋值
    p = &i

    // 通过指针变量赋值
    *p = 200

    // 获取100
    // 通过变量i获取
    fmt.Println(i)
    fmt.Println(*p)    
}

注意

  • 在使用指针变量p来修改变量i的值的时候，前面一定要加上*（通过指针访问目标对象）

4.3 结论

  • 定义变量时*表示类型是指针类型，没有其他的用处
  • 获取指向的空间中的数据时，使用*指针变量名

5. 注意事项

在使用指针变量时，要注意以下两点

5.1 默认值为nil

package main

import "fmt"

func main()  {

    // 定义指针变量，默认为nil
    var p *int

    // 打印
    fmt.Println(p)    
}

5.2 不要操作没有合法指向的内存

例如，在上面的案例中，我们定义了指针变量p，但是没有让指针变量指向任何一个变量，那么直接运行如下程序，会出现异常。

package main

import "fmt"

func main()  {

    // 定义指针变量，默认为nil
    var p *int

    // 打印
    fmt.Println(p)

    // 给p指向的空间设置为100
    *p = 100    
}

注意：

  • 在使用指针变量时，一定要让指针变量有正确的指向

以下的操作是合法的：

package main

import "fmt"

func main()  {

    // 定义指针变量，默认为nil
    var p *int
    // 定义普通变量
    var num int

    // 让p指向num这个空间
    p = &num

    // 给p指向的空间设置为100
    *p = 100

    // 获取数据
    fmt.Println(*p)    
}

说明

  • 只要保证指针变量指向的是自己定义的变量的空间（即合法空间），那么就不会有错误

6. new()函数

指针变量，不仅可以指向我们自定义的变量空间，还可以指向通过new( )函数创建的空间

具体的应用方式如下:

package main

import "fmt"

func main()  {

    // 定义指针变量，默认为nil
    var p *int

    // 创建一个空间，让变量p指向
    p = new(int)

    // 给p指向的空间赋值
    *p = 100

    // 取值
    fmt.Println(*p)
}

说明

  • new(int)作用就是创建一个整型大小的空间
  • 指针变量p指向了new()创建的空间，所以通过指针变量p进行赋值后，该空间中的值就是100
  • new( )函数的作用就是C语言中的动态分配空间。但是在这里与C语言不同的地方是：Go语言不需要管理该空间的释放，GO语言会自动释放。这也是比C语言使用方便

也可以使用自动推导类型的方式：

package main

import "fmt"

func main()  {
    // 创建一个空间，让指针变量p指向
    p := new(int)

    // 给p指向的空间赋值
    *p = 100

    // 取值
    fmt.Println(*p)
}

7. 指针做函数参数

指针也可以作为函数参数

我们知道，当把int类型的变量当做实参时，实际上会将数据的备份，即数值，进行了传递

如果我们需要将int类型变量的地址传递，就可以将这个变量的指针传递即可，方式如下：

package main

import "fmt"

func Test(tempA, tempB *int)  {
    // 交互2个空间中的数据
    *tempA, *tempB = *tempB, *tempA
}

func main()  {
    var a int = 100
    var b int = 200

    Test(&a, &b)

    fmt.Printf("在main函数中a=%d, b=%d", a, b)
}

8. 数组指针

8.1 什么是数组指针

一个地址，不是普通的变量的地址，而是一个数组的地址，我们把这样的指针（地址），称之为 数组指针

通俗点说：一个可以指向数组的指针

8.2 定义方式

var 指针变量名 *[数组元素个数]类型

8.3 demo

package main

import "fmt"

func main()  {
    nums := [3]int {11, 22, 33}
    var p *[3]int
    // 让p指向数组
    p = &nums

    // 通过指针操作数组
    fmt.Println(p)
    fmt.Println(*p)
    fmt.Println((*p)[0])
    fmt.Println((*p)[1])

    // 通过指针修改数组中的数据
    (*p)[2] = 55

    fmt.Println((*p)[2])
}

8.4 demo2

package main

import "fmt"

func Test(temp *[3]int)  {
    fmt.Println((*temp)[0])
    fmt.Println((*temp)[1])
    fmt.Println((*temp)[2])

    fmt.Println("-------------")

    for _, num := range *temp {
        fmt.Println(num)
    }
}

func main()  {
    nums := [3]int {11, 22, 33}
    // 获取数组的指针的方式 与获取普通变量的指针的方式一样，添加&
    Test(&nums)
}

9. 指针数组

9.1 什么是指针数组

指针数组：一个数组中存储的都是指针（也就是地址）

通俗的说：就是一个存储了地址的数组

9.2 定义方式

var 数组变量名 [元素个数]*类型

9.3 demo1

package main

import "fmt"

func main()  {
    // 定义一个指针数组
    var p [3]*int

    // 定义3个变量
    a := 11
    b := 22
    c := 33

    // 给数组元素赋值
    p[0] = &a
    p[1] = &b
    p[2] = &c

    // 打印a的地址
    fmt.Println(p[0])
    // 打印a中存储的数据
    fmt.Println(*p[0])
}

9.4 demo2

package main

import "fmt"

func main()  {
    // 定义一个指针数组
    var p [3]*int

    // 定义3个变量
    a := 11
    b := 22
    c := 33

    // 给数组元素赋值
    p[0] = &a
    p[1] = &b
    p[2] = &c

    // 使用for循环遍历
    for _, temp := range p{
        // 因为temp是指针，所以*temp才能提取指向的空间中的数据
        fmt.Println(*temp)
    }
}

10. 结构体指针

10.1 什么是结构体指针

指针指向了数组，解决了数组引用传递的问题。那么指针指向结构体，是否能够解决结构体引用传递的问题呢？

完全可以

通俗点说：一个指针指向了结构体 就是 结构体指针

10.2 定义格式

var 结构体指针变量名 *结构体类型

10.3 demo1

方式1

package main

import "fmt"

func main()  {

    // 定义结构体类型
    type Student struct{
        id int
        name string
    }

    // 定义结构体指针变量
    var p *Student = &Student{101, "东哥"}

    // 通过结构体指针设置成员变量的值
    p.id = 102


    // 通过结构体指针获取成员变量的值
    fmt.Println(p.id)
    fmt.Println(p.name)
}

方式2：自动推导类型

package main

import "fmt"

func main()  {

    // 定义结构体类型
    type Student struct{
        id int
        name string
    }

    // 定义结构体指针变量
    p := &Student{101, "东哥"}

    // 通过结构体指针设置成员变量的值
    p.id = 102


    // 通过结构体指针获取成员变量的值
    fmt.Println(p.id)
    fmt.Println(p.name)
}

10.4 demo2

将结构体指针当做实参

package main

import "fmt"

// 定义结构体类型
type Student struct{
    id int
    name string
}

func Test(temp *Student)  {
    temp.id = 102
    temp.name = "老王"
}

func main()  {



    // 定义结构体指针变量
    p := &Student{101, "东哥"}

    // 调用Test函数，将p当做实参
    Test(p)

    // 通过结构体指针获取成员变量的值
    fmt.Println(p.id)
    fmt.Println(p.name)
}
