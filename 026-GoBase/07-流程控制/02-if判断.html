2. if

在编程中实现选择判断结构就是用if

结构基本语法如下：

if 条件判断 {
    代码语句
}

条件判断

  • 如果为真（true）,那么就执行大括号中的语句

  • 如果为假（false），就不执行大括号中的语句

具体案例如下：

package main

import "fmt"

func main()  {
    var age int = 20
    var result bool

    // 将age > 18的结果，即要么是true 要么 是false 存到result变量中
    result = age > 18

    // 使用if判断条件是否是true
    if result {
        // 如果以上的判断条件成立，那么就会执行{ }中的代码
        fmt.Printf("年龄是%d,已经成年了", age)
    }
}

将上面的程序进行简化：

package main

import "fmt"

func main()  {
    var age int = 20
    // var result bool

    // 将age > 18的结果，即要么是true 要么 是false 存到result变量中
    // result = age > 18

    // 使用if判断条件是否是true
    // if result {
    if age > 18 {
        // 如果以上的判断条件成立，那么就会执行{ }中的代码
        fmt.Printf("年龄是%d,已经成年了", age)
    }
}

条件不满足时不执行

package main

import "fmt"

func main()  {
    var age int = 20

    if age > 30 {
        // 如果以上的判断条件成立，那么就会执行{ }中的代码
        fmt.Printf("年龄是%d,已经成年了", age)
    }
}

3. if else

想一想：

在使用if的时候，它只能做到满足条件时要做的事情，那万一需要在不满足条件的时候，做某些事，该怎么办呢？例如实际生活中，年龄60岁以上则免费，否则就收费，就需要根据条件是否满足做出不同的功能

基本语法结构如下：

if 条件判断 {
    条件成立执行的代码1
    条件成立执行的代码2
    条件成立执行的代码3
      。。。
} else {
    条件不成立执行的代码1
    条件不成立执行的代码2
    条件不成立执行的代码3
      。。。
}

执行流程：

  1. if对条件进行判断，看是否为true，如果是true则执行 条件成立执行的代码1、2、3
  2. 如果条件为false则执行 条件不成立执行的代码1、2、3
  3. 无论条件知否成立，执行完对应的功能后，就表示整个if语句结束了，程序会继续向下运行

if-else案例

小赵的考试成绩大于90(含)分,那么爸爸奖励他100元钱,否则的话,爸爸就让小赵跪方便面。

有同学可能会想，这道题用if结构也可以实现。如下所示：

package main

import "fmt"

func main()  {
    // 要求：小赵的考试成绩大于90(含)分,那么爸爸奖励他100元钱,否则的话,爸爸就让小赵跪方便面。

    // 1. 定义变量，用来存储分数
    var score int

    // 2. 从键盘获取数据，存入到score中
    fmt.Scanf("%d", &score)

    // 3. 使用if判断是否是90分
    if score >= 90 {
        // 3.1 如果是，则奖励100元
        fmt.Println("奖励100元")
    } else {
        // 3.2 如果不是，则惩罚跪方便面
        fmt.Println("跪方便面")
    }
}

4. if else if

上面我们学习了if以及if-else的使用，简单总结一下

  • if

    当需要根据条件满足或者不满足时，满足1种条件执行

  • if-else

    当条件满足时、不满足时 都需要做事情，满足2种条件执行

试想下，真实情况下，有时需要多个条件的判断，例如 参观某个景点时，成人收费，老人免费，儿童120cm以下免费，120cm-150cm半票，150cm以上全票等，就需要用到我们本节的知识if else if

基本语法如下：

package main

import "fmt"

func main()  {
    if 条件判断1 {
        要执行的代码段1
    } else if 条件判断2 {
        要执行的代码段2
    } else if 条件判断3 {
        要执行的代码段3
    } else if 条件判断4 {
        要执行的代码段4
    } else{
        如果上述的所有if判断条件都不满足，则执行这里的代码
    }        
}

执行流程：

  1. 如果条件1成立，即结果为true那么，就执行 要执行的代码段1，然后整个if全部结束
  2. 如果条件1不成立，那么判断 条件2是否成立，如果条件2成立，即结果为true，那么就执行 要执行的代码段2，然后整个if全部结束
  3. 如果条件2不成立，则判断条件3.。。。。。。
  4. 如果条件1、2、3、4都不成立，则执行else对应的{ }中的代码，然后整个if全部结束

if else if案例

package main

import "fmt"

func main()  {
    // 要求：从键盘输入一个1-7之间的数字，然后输出其对应的星期几，例如1输出星期一 2输出星期二...

    // 1. 定义一个变量，用来存储用户输入的数字
    var num int

    // 2. 从键盘输入数据，且存储到num变量中
    fmt.Scanf("%d", &num)

    // 3. 使用if判断，其对应的星期几
    if num == 1 {
        fmt.Println("星期一")
    } else if num == 2 {
        fmt.Println("星期二")
    } else if num == 3 {
        fmt.Println("星期三")
    } else if num == 4 {
        fmt.Println("星期四")
    } else if num == 5 {
        fmt.Println("星期五")
    } else if num == 6 {
        fmt.Println("星期六")
    } else if num == 7 {
        fmt.Println("星期日")
    } else {
        fmt.Println("输入有误...")
    }        
}

5. if嵌套

什么是嵌套

大家是否见过套娃，如下图

所谓的if嵌套，就是if中的{ }里面还有if

能够实现的功能，例如：丈母娘问你有房吗？你说有，然后丈母娘问，你有车吗？你说有，然后丈母娘接着问，你有存款吗？你说有，然后丈母娘说女儿嫁给你 😃😃😃

if嵌套案例

输入公交卡当前的余额，只要超过2元，就可以上公交车；如果空座位的数量大于0，就可以坐下

package main

import "fmt"

func main()  {
    // 要求：输入公交卡当前的余额，只要超过2元（包含等于），就可以上公交车；如果空座位的数量大于0，就可以坐下
    // 1. 定义一个变量用来存储当前的公交卡的余额
    // 2. 从键盘获取余额
    // 3. 判断是否超过2元
    // 3.1 如果是 则 获取空座位的数据
    //              然后判断是否大于0
    //              如果是，则显示可以坐下
    //              如果不是，则显示只能站着
    // 3.2 如果不是 则 不能上公交车

    var money int
    fmt.Println("请输入你公交卡的余额：")
    fmt.Scan(&money)
    if money >= 2 {
        fmt.Println("可以上公交车了...")
        var seat_count int
        fmt.Println("请输入当前剩余的座位数：")
        fmt.Scan(&seat_count)
        if seat_count > 0 {
            fmt.Println("可以坐下休息...")
        } else {
            fmt.Println("没有座位了，只能站着...")
        }
    } else {
        fmt.Println("不好意思，余额不足，不能上公车")
    }
}
