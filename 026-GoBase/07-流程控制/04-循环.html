1. 循环概念

所谓的循环就是重复的做同一件事情，我们先从生活的角度来理解一下循环。

在编程中，我们也会经常遇到循环的问题，例如：

跟媳妇承认错误，说一万遍"媳妇儿，我错了"


fmt.Println("媳妇儿，我错了")

fmt.Println("媳妇儿，我错了")

fmt.Println("媳妇儿，我错了")

...省略...

2. 基本语法结构

在GO语言中，使用for能够完成循环

注意：

GO语言中只有for循环结构，没有while, do-while结构

基本语法结构如下：

for 表达式1;表达式2;表达式3 {
    // 这里是循环体，就是要执行的代码
    要重复执行的代码1
    要重复执行的代码2
    要重复执行的代码....
}

说明:

  • 表达式1：定义一个循环的变量，记录循环的次数
  • 表达式2：一般为循环条件，循环多少次
  • 表达式3：一般为改变循环条件的代码，使循环条件终有一天不再成立
  • 循环体：重复要做的事情

下面我们用for循环结构输出10遍"媳妇儿，我错了"。

package main

import "fmt"

func main() {
    var num int
    for num=1; num<=10; num ++ {
        fmt.Println("媳妇我错了")
    }

}

执行顺序说明:

  1. 执行序号1 一次
  2. 执行序号2
    1. 如果结果为true，则执行 序号4、3，然后再次执行序号2，如果结果是true那么就再执行4、3，直到当某次序号2结果为false时就结束for循环
    2. 如果结果为false，直接结束for循环

4. 跳转语句

关于GO语句中的跳转语句，有

  • break
  • continue
  • goto(与函数结合使用，后期讲解到函数在讲)

4.1 break

package main

import "fmt"

func main() {

    for a:=1; a<=5; a ++ {
        fmt.Printf("a=%d\n", a)
    }

    fmt.Println("----------------")

    for b:=1; b<=5; b ++ {
        fmt.Printf("b=%d", b)
        break  // break的作用是退出其所在的for循环，即结束for循环
    }
}

4.2 break案例

需求：

要求用户输入用户名和密码，默认的用户名为admin，密码是888888，如果用户输入错误则提醒用户重新输入，总共3次，如果3次都不正确，则提示用户名或密码错误

参考代码：

package main

import "fmt"

func main() {
    // 需求：要求用户输入用户名和密码，默认的用户名为admin，密码是888888，
    // 如果用户输入错误则提醒用户重新输入，总共3次，如果3次都不正确，则提示用户名或密码错误

    // 流程
    // 1. 定义变量，用来存储用户名以及密码
    // 2. 使用for循环重复执行代码(3次)
    // 2.1 从键盘获取用户名以及密码
    // 2.2 for循环体中 使用if对用户名以及密码与默认的进行对比
    // 2.3 如果正确则提示：登录成功
    // 2.4 如果不正确则提示：用户名或密码错误，请重新输入

    var user_name string
    var user_passwd string

    for i:=1; i<=3; i++ {
        fmt.Println("请输入用户名:")
        fmt.Scanf("%s", &user_name)
        fmt.Println("请输入密码:")
        fmt.Scanf("%s", &user_passwd)
        if user_name == "admin" && user_passwd == "888888" {
            fmt.Println("登录成功")
            break
        } else {
            fmt.Println("用户名或密码错误，请重新输入")
        }
    }
}

4.3 continue

continue的作用是立即结束本次循环,然后判断条件是否成立然后进入下一次循环,否则退出循环

基本使用方式：

package main

import "fmt"

func main() {

    for b:=1; b<=5; b ++ {
        fmt.Printf("b=%d\n", b)
        continue
        fmt.Printf("哈哈哈1")
        fmt.Printf("哈哈哈2")
    }
}

通常我们使用的方式：

package main

import "fmt"

func main() {

    for b:=1; b<=5; b ++ {
        fmt.Printf("b=%d\n", b)
        if b == 3 {
            continue
        }
        fmt.Printf("哈哈哈1\n")
        fmt.Printf("哈哈哈2\n")
    }
}

continue案例

需求：

用continue实现计算1到100(含)之间的除了能被7整除的所有数的和

package main

import "fmt"

func main() {
    // 方式1（未使用continue）
    sum := 0
    for i:=1; i<=100; i++ {
        if i % 7 == 0 {
            sum += i
        }
    }
    fmt.Printf("1-100之间能被7整除的所有数的和是:%d\n", sum)

    // 方式2（使用continue）
    sum = 0  // 需要将sum的值重置为0
    for i:=1; i<=100; i++ {
        if i % 7 != 0 {
            continue
        }
        sum += i
    }
    fmt.Printf("1-100之间能被7整除的所有数的和是:%d\n", sum)
}

5. 循环嵌套

我们在前面学习选择结构时，讲解过关于选择结构的嵌套，同理循环结构也有嵌套。

一句话描述：for循环体中又包含for循环

5.1 基本格式

for 初始化值1;条件1;自增1 {

      循环体1......

    for 初始化值2;条件2;自增2 {
        循环体2......
    }
}

特点：

  • 外层循环每执行一次，内层循环执行所有的循环

5.2 循环嵌套案例

需求

  1. 用for循环嵌套的方式实现
  2. 总共说3次如下的话
    1. 先说一句：亲爱的老婆大人
    2. 然后说三句：我的所有

参考代码

package main

import "fmt"

func main() {
    // 需求
    // 1. 用for循环嵌套的方式实现
    // 2. 总共说3次如下的话
    //         2.1 先说一句：亲爱的老婆大人
    //         2.2 然后说三句：我的所有

    // 外层循环用来控制整体的次数
    for i:=1; i<=3; i++ {
        fmt.Println("亲爱的老婆大人")
        // 内存循环用来控制 打印3次 我的所有
        for j:=1; j<=3; j++ {
            fmt.Println("我的所有")
        }
    }
}

无限循环，从名字就能看出，它一直循环，永不停息

我们也把这种循环，称之为 死循环

死循环结构如下

for {
      循环体...
}

特点:

  • 没有各种条件以及个表达，只有for

实际案例

前面我们使用for循环做过一个案例

要求用户输入用户名和密码，默认的用户名为admin，密码是888888，如果用户输入错误则提醒用户重新输入，总共3次，如果3次都不正确，则提示用户名或密码错误

如果，现在将需求进行修改，即不限次数的让用户登录输入用户名、密码，那该怎么做呢？

package main

import "fmt"

func main() {
    var user_name string
    var user_passwd string

    // for i:=1; i<=3; i++ {  // 这是原来的方式，最多执行3次登录
    for {  // 这是实际需求，只要用户名不正确，可以紧接着再次尝试，知道登录成功为止
        fmt.Println("请输入用户名:")
        fmt.Scanf("%s", &user_name)
        fmt.Println("请输入密码:")
        fmt.Scanf("%s", &user_passwd)
        if user_name == "admin" && user_passwd == "888888" {
            fmt.Println("登录成功")
            break
        } else {
            fmt.Println("用户名或密码错误，请重新输入")
        }
    }
}
