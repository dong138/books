switch结构

1. 引入

在讲解switch结构之前，我们先来看如下问题：

李四的年终工作评定,如果定为A级,则工资涨500元,如果定为B级,则工资涨200元,如果定为C级,工资不变,如果定为D级工资降200元,如果定为E级工资降500元.

设李四的原工资为5000,请用户输入李四的评级,然后显示李四来年的工资.

大家看到如上题目，肯定想到的是用if else if结构来实现，那么具体案例实现如下：

package main

import "fmt"

func main()  {
    // 需求：
    // 李四的年终工作评定,如果定为A级,则工资涨500元,如果定为B级,则工资涨200元,如果定为C级,工资不变,如果定为D级工资降200元,如果定为E级工资降500元.
    // 设李四的原工资为5000,请用户输入李四的评级,然后显示李四来年的工资

    var salary int = 5000
    var level byte
    fmt.Printf("请输入级别（A、B、C、D、E）：")
    fmt.Scanf("%c", &level)
    if level == 'A' {
        salary += 500
    } else if level == 'B' {
        salary += 200
    } else if level == 'C' {
        salary += 0
    } else if level == 'D' {
        salary -= 200
    } else if level == 'E' {
        salary -= 500
    } else {
        fmt.Println("输入有误，程序退出...")
    }

    fmt.Printf("评定级别为:%c, 工资调整为:%d\n", level, salary)
}

运行效果：

虽然以上代码实现了我们的需求，但是我们发现了一个问题，如果输入其它字符（例如：输入字符T），我们发现最终的结果不仅输出了"输入错误，程序退出"，同时还输出了"李四来年的工资是：5000"。如果输入错误了，这句话是不应该输出的，那么应该怎样进行处理呢？

package main

import "fmt"

func main()  {
    // 需求：
    // 李四的年终工作评定,如果定为A级,则工资涨500元,如果定为B级,则工资涨200元,如果定为C级,工资不变,如果定为D级工资降200元,如果定为E级工资降500元.
    // 设李四的原工资为5000,请用户输入李四的评级,然后显示李四来年的工资

    var salary int = 5000
    var level byte
    var error_flag bool
    fmt.Printf("请输入级别（A、B、C、D、E）：")
    fmt.Scanf("%c", &level)
    if level == 'A' {
        salary += 500
    } else if level == 'B' {
        salary += 200
    } else if level == 'C' {
        salary += 0
    } else if level == 'D' {
        salary -= 200
    } else if level == 'E' {
        salary -= 500
    } else {
        error_flag = true
        fmt.Println("输入有误，程序退出...")
    }

    if !error_flag {
        fmt.Printf("评定级别为:%c, 工资调整为:%d\n", level, salary)
    }

}

我们前面在讲解if else if时讲过，它适合什么场景？

  1. 多条件判断
  2. 区间性的数据判断

但是我们看一下该案例，是否是对区间性的数据进行判断呢？不是，而是定值判断，也就是对一个固定值的判断。

对这种固定值的判断推进使用switch-case结构

2. switch

语法

switch 变量或者表达式的值 {
    case 值1:
        要执行的代码
    case 值2:
        要执行的代码
    case 值3:
        要执行的代码
    default:
        要执行的代码
}

switch-case结构的执行流程：

  1. 首先将变量或者表达式的值计算出来
  2. 拿着这个值依次跟每个case后面所带的值进行匹配
  3. 一旦匹配成功，则执行该case所带的代码，执行完成后，立即结束整个switch语句
  4. 如果，跟每个case所带的值都不匹配。如果有default，则执行default中的语句，如果没有default，则什么都不做

案例

package main

import "fmt"

func main()  {
    // 需求：从键盘输入1个月份的数字，例如1，然后输出对应是几月份
    var month int

    fmt.Scanf("%d", &month)

    switch month {
        case 1: fmt.Println("1月")
        case 2: fmt.Println("2月")
        case 3: fmt.Println("3月")
        case 4: fmt.Println("4月")
        case 5: fmt.Println("5月")
        case 6: fmt.Println("6月")
        case 7: fmt.Println("7月")
        case 8: fmt.Println("8月")
        case 9: fmt.Println("9月")
        case 10: fmt.Println("10月")
        case 11: fmt.Println("11月")
        case 12: fmt.Println("12月")    
        default: fmt.Println("输入有误....")
    }

}

注意点

GO语言中switch-case结构中，case后面默认自带break(终止程序执行)，这是与其它语言不同的地方。

注意：

如果我们想执行完成某个case后，强制执行后面的case,可以使用fallthrough

3. if else if vs switch

其实通过上面的案例，我们发现switch能够实现的，我们用if else if也可以实现，那么反过来呢，if else if能实现的，我们使用switch能否实现呢？接下来将我们前面用if-else if实现的"考试成绩评定"这道题用switch来实现。

package main

import "fmt"

func main()  {
    // 需求：输入分数，输出对应等级
    var score int

    fmt.Println("请输入本次考试的分数:")
    fmt.Scanf("%d", &score)

    switch {
        case score >= 90 && score <= 100 : fmt.Println("等级为A") 
        case score >= 80: fmt.Println("等级为B") 
        case score >= 70: fmt.Println("等级为C") 
        case score >= 60: fmt.Println("等级为D") 
        case score >= 0: fmt.Println("等级为E") 
        default: fmt.Println("输入有误....")
    }
}

总结

  • 相同点

    都可以实现多分支结构

  • 不同点（根据个人习惯进行选择）

    if else if:可以处理范围

    switch:一般用于等值比较

4. 综合应用

需求：

请用户输年份,再输入月份,输出该月的天数.(结合之前如何判断闰年来做)

参考代码：

package main

import "fmt"

func main() {
    //请用户输年份,再输入月份,输出该月的天数.
    var year int  // 表示年份
    var month int  // 表示月份
    var day int

    fmt.Println("请输入年份")
    fmt.Scanf("%d", &year)
    fmt.Println("请输入月份")
    fmt.Scanf("%d", &month)

    // 判断月份的取值范围
    if month >=1 && month <= 12{
        switch month {
            case 1:
                fallthrough
            case 3:
                fallthrough
            case 5:
                fallthrough
            case 7:
                fallthrough
            case 8:
                fallthrough
            case 10:
                fallthrough
            case 12:
                day = 31
            case 2: // 需要判断是否为闰年
                if (year % 400 == 0) || (year % 4==0 && year % 100 != 0){
                    day = 29
                }else{
                    day = 28
                }
            default:
                day = 30
        }
        fmt.Printf("%d年%d月共%d天\n", year, month, day)
    } else {
        fmt.Println("月份输入错误！！")
    }
}

心得：

通过上面的案例，我们发现switch和if-else是可以结合来使用的，可以根据具体的问题具体分析，灵活来应用。

fallthrough

Go里面switch默认相当于每个case最后带有break，匹配成功后不会自动向下执行其他case，而是跳出整个switch, 但是可以使用fallthrough强制执行后面的case代码。
