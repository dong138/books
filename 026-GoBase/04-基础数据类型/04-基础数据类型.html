Go基础

基础数据类型

前面我们写的程序中，用到的变量的类型都是整型，但是在我们GO语言中，还有很多其它的类型

下面我们看一下几个非常重要的类型，其它类型随着课程的深入，后面会给大家讲解到。

1. 布尔类型

在计算机中,我们用什么数据类型来表示一句话的对错，例如：在计算机中描述张三(20岁)比李四(18)小,这句话的结果？

布尔类型的变量取值要么是真（true）,要么是假(false),用bool关键字来进行定义。示例如下：

package main

import "fmt"

func main()  {
    // 定义一个变量a，是布尔类型的，如果不初始化指定数据，默认是false
    var a bool
    fmt.Println("a = ", a)

    // 自动识别类型
    var b = true
    fmt.Println("b = ", b)

    // 自动识别类型，且不使用var
    c := false
    fmt.Println("c = ", c)
}

指定格式的输出：

package main

import "fmt"

func main()  {
    var d = false
    fmt.Printf("d = %t", d)
}

2. 浮点型

package main

import "fmt"

func main()  {
    // 定义一个变量是浮点类型的，给其赋初始值3.14
    var pi float32 = 3.14
    fmt.Println("pi = ", pi)

    // 自动推到类型
    pi2 := 1.667
    fmt.Println("pi2 = ", pi2)
}

有格式的输入与输出

package main

import "fmt"

func main()  {
    var height float32
    // 获取用户的身高
    fmt.Scanf("%f", &height)

    // 输出用户的身高
    fmt.Println("用户的身高是", height)
}

注意：

// 会推到为fload类型
var pi
pi=3.14

// 如果不加小数点，f2会被推导为整型而不是浮点型
f2:=12.0

3. 字符类型

3.1 字符基本输入与输出

所谓字符类型是用单引号括起来的单个字符，关于字符的定义使用如下：

package main

import "fmt"

func main()  {
    // 声明一个变量是字符类型
    var ch byte
    // 使用单引号包裹起来的表示字符类型的值
    ch = 'a'
    // 输出字符类型
    fmt.Printf("ch=%c", ch)
}

3.2 输入一个字符

如下所示：

package main

import "fmt"

func main()  {
    // 声明一个变量是字符类型
    var ch byte
    // 获取键盘输入的字符
    fmt.Scanf("%c", &ch)
    // 输出字符
    fmt.Printf("用户刚刚输入的是%c\n", ch)
}

3.3 字符第二种输入与输出方式

package main

import "fmt"

func main()  {
    // 声明一个变量是字符类型
    var ch byte
    // 给ch一个数字
    ch = 97
    // 输出字符
    fmt.Printf("数字97对应的字符是：%c\n", ch)
}

这里定义了，ch是一个字符类型，赋值却是一个整数97，打印的结果是小写字符'a'.

原因是：计算机不能直接存储字符类型，只能转成数字存储，但是为什么小写字符'a',对应的整数是97呢？因为，计算机是根据'ASCII'码来存储。（这里只是简单介绍ASCII）

注意强调：

package main

import "fmt"

func main()  {
    // 声明一个变量是字符类型
    var ch byte
    // 给ch一个数字
    ch = 97
    // 输出字符
    fmt.Printf("对数字97按照字符方式打印：%c\n", ch)
    fmt.Printf("对数字97按照整型方式打印：%d\n", ch)
}

问题，以下代码输出的结果是多少？

    // 声明一个变量是字符类型
    var ch byte
    // 给ch一个字符值
    ch = 'b'
    // 输出字符

    fmt.Printf("按照整型方式打印：%d\n", ch)
}

3.4 转义字符

字符除了我们常见的字符'a',字符'b'等等以外，还有一些特殊的字符，例如我们讲解的'\n', 它的作用是什么？换行，不会在屏幕上输出具体的内容。这种字符我们称为转义字符（具有特殊含义），那么除了'\n'字符以外，还有哪些是我们以后编程中经常用到的转义字符呢？

\":表示一个英文半角的双引号

package main

import "fmt"

func main()  {
    fmt.Println("欢迎来到 \"通俗易懂学编程\" 欢迎大家")
}

\t:表示一个tab键的空格（可以用来对输出的内容进行排版）

package main

import "fmt"

func main()  {
    fmt.Println("姓名\tQQ号\t省份")
    fmt.Println("王老师\t10086\t山东")
    fmt.Println("李老师\t10010\t北京")
}

\\:表示一个\(主要用在表示磁盘目录上，后面我们讲解文件操作时，经验要根据磁盘目录获取文件然后进行读取，写入等操作)

package main

import "fmt"

func main()  {
    fmt.Println("D:\\王老师\\课件\\Go语言")
}

以上就是我们以后编程中经常用到的转义字符，需要大家记住。

4. 字符串类型

用单引号括起来的单个字符是字符类型，用双引号括起来的字符是字符串类型

下面是关于字符串的定义使用：

package main

import "fmt"

func main()  {
    var url string  // 声明一个变量，其类型是字符串
    url = "howdoit.cn"  // 给url变量赋值
    fmt.Printf("我们的网址是:%s", url)  // 打印字符串时使用%s
}

字符串输入：

package main

import "fmt"

func main()  {
    var name string
    // 获取用户的姓名
    fmt.Scanf("%s", &name)
    // 输出用户姓名
    fmt.Printf("报名参加学习Go语言的学生姓名是:%s\n", name)
}

5. 字符与字符串区别

关于字符与字符串的区别：

package main

import "fmt"

func main()  {
    // 定义字符时，使用byte类型
    var gender byte
    // 定义字符串时，使用string类型
    var name string

    // 字符
    // 一对单引号组成
    // 往往在单引号中只有1个字符，当然如果需要转义的话，就2个，例如'\n'
    gender = 'F'
    fmt.Printf("性别是:%c\n", gender)

    // 字符串
    // 一对双引号组成
    // 往往由1个或者多个字符组成
    // 字符串会在末尾隐藏一个\0，它用来表示字符串结束
    name = "wanglaoshi"  // 其实末尾还有一个\0 即 wanglaoshi\0，但是\0我们不用写

    // 如果想要测试字符串的长度，可以使用len
    fmt.Println("字符串name的长度是:", len(name))
}
